IDE Integration

Use XProxy inside your editor — not a separate window.

VS Code Copilot Chat

Route Copilot Chat through XProxy via the LLM Gateway extension.

  1. Install GitHub Copilot LLM Gateway
  2. Open Copilot Chat → model picker → Select a language model
  3. Click ⚙️ → Add models → LLM Gateway
  4. Enter your XProxy URL and vscode as the API key
  5. Select your model from the picker

Terax

Point Terax at XProxy with a bearer token.

  1. Set API base to http://localhost:8000/v1
  2. Set Authorization: Bearer terax
  3. Model name matches your chosen provider

Quick start

1

Clone & install

Node.js required. No build step — just npm install.

2

Paste a fetch

Copy any fetch() from your browser DevTools on DeepSeek, ChatGPT, or Claude. Paste into the wizard.

3

Point your IDE

Set your IDE to http://localhost:8000/v1. Start chatting like a real browser user.

~/xproxy · cmd 
# 1. clone, install, start
>git clone https://github.com/downloaddoctor/xproxy.git
>cd xproxy
>npm install && npm start

# 2. interactive wizard — pick provider, paste fetch, choose session

# 3. call it like OpenAI
>curl http://localhost:8000/v1/chat/completions \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer vscode" \
    -d '{"messages":[{"role":"user","content":"hello"}],"model":"deepseek"}'
